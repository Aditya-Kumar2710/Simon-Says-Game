# 🎮 Simon Says Game

A fun and interactive memory-based game built using HTML, CSS, and JavaScript. Test your memory skills by repeating the sequence of colors correctly as the game progresses!

# 🌐 Live Demo:
👉 https://simonsaysbyadi.netlify.app/

# 📌 Features

##🎯 Interactive gameplay with smooth UI

##🧠 Improves memory and focus

##🔁 Increasing difficulty with each level

##❌ Game over detection with restart option

##⚡ Fast and responsive design

#🛠️ Tech Stack

##HTML5 – Structure of the game

##CSS3 – Styling and animations

##JavaScript (Vanilla JS) – Game logic and interactivity

#🎮 How to Play

Press any key to start the game

Watch the sequence of flashing colors carefully

Repeat the sequence by clicking the buttons in the same order

Each correct round adds a new step to the sequence

If you click the wrong button → Game Over

Try to beat your highest score 🚀

#💡 Game Logic (Concept)

The game works on a simple idea:

A random color sequence is generated

The sequence increases every level

User input is checked step-by-step

If input mismatches → game resets

##This type of game is widely used to test pattern recognition and memory skills
